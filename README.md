PawnStar

PawnStar is a browser-based chess platform where players can compete in real time while video chatting, and an audience can watch the game live, comment on moves, and engage with the players.





Features :

Real-time multiplayer chess

Peer-to-peer video calling for players (WebRTC)

Spectator mode with live board updates

Audience chat and engagement during games

Room-based play via shareable links

Legal move validation using chess.js

Real-time synchronization with WebSockets
